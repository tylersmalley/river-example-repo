# river-example-repo

A minimal Go "Hello, World!" application used to exercise River sandboxes.

## Tasks

This repo uses [Mise](https://mise.jdx.dev) for toolchain and task management.

- `mise run setup` — install dependencies
- `mise run run` — run the program
- `mise run build` — build the binary
- `mise run test` — run tests
- `mise run check` — run tests, formatting, and vet checks
